Introduction to Android

By Dillon Grove and Brandon Kase

In this intro-level Android workshop, we'll make an application that displays weather information based on a user-provided zip code. In order to do this, we'll cover how to create an application with multiple activities, how to pass information between activities, how to scrape data from a webpage, and how to display it to the user. Note: this talk was given at TartanHacks 2012 as well so you may be bored if you already attended that one!
