Backbone.js

Ilter Canberk

Description

This talk will be about client-side heavy web apps, and maintaining a large Javascript codebase in general. The first part will be more like a discussion of current technologies out there and why/how we made some of the technical decisions while building Appcubator. For the second part, we will learn about Backbone.js and build a demo together.

Summary of the topics are below.

  • What is a client-side web app? What are the advantages and disadvantages as opposed to other approaches?
  • Overview of all the current frameworks out there, and their comparison
  • Introduction to Backbone.js in more detail
  • Writing modular javascript
  • Using require.js to control module dependency

For the demo, we will build a simple online version of MS Paint. I will be providing a lot of the boilerplate code and some of the modules.

Times

7:00pm

Prerequisites

Participants should be familiar with:

  1. Javascript in general
  2. jQuery library and CSS selectors
  3. Some of the software design patterns such as inheritance
