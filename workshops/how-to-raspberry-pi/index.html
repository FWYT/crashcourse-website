How To: Raspberry Pi

Brent Strysko

Description

Ever see a camera detect a person and then have a nurf gun turn in their direction to shoot at them? What about those cool looking internet accessible lighting strips you have only seen on tv? If you ever wondered how any of these "hardware hacks" work this is the talk for you. Using the Raspberry PI as an embedded Linux solution we will look at some of the cool applications you can implement. The first half of the talk will consist of going over some hardware fundamentals and how to get a Raspberry Pi up and going. The latter half of the talk will be Q&A about some popular hardware hacks and how to get started with hardware hacking in general and specifically at CMU.

Times

1:30pm

Prerequisites

  • Basic Linux Bash (cd, cat, echo)
  • Rudimentary understanding of electricity (what voltage, current, resistance, and Ohm's Law are)
