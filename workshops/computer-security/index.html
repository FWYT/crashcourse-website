Computer Security by PPP

Maxime Serrano & Ryan Goulden

Description

Ryan Goulden and Maxime Serrano are two of the leaders of PPP, CMU's competitive computer security team, which consistently ranks as one of the best in the world. This CrashCourse will cover basic web vulnerabilities, including SQL injections, cross-site scripting and some other vulnerabilities that are common on the web. After a short discussion about each kind of vulnerability, attendees will have an opportunity to execute the attacks themselves on a custom-made vulnerable site. We will then discuss good ways to avoid introducing these vulnerabilities into your code, and if time permits discuss other kinds of security issues, such as cryptography and other forms of exploitation.

Attendees should bring a laptop either with Linux or the ability to connect into the Andrew Unix machines, be familiar with a text editor, and have some basic familiarity with the terminal. The first three non-syllabus lectures from the Great Practical Ideas course should be plenty.
